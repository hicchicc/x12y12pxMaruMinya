# x12y12pxMaruMinya
MaruMinya (マルミーニャ) is a cute and versatile rounded Japanese pixel font. under SIL OFL 1.1

- Version 1.500 は、従来の独自ライセンス版データを元にSIL OFLライセンス移行のみを行った、応急的なバージョンです
- 派生フォントである [x12y12pxMaruMinyaHangul](https://github.com/quiple/x12y12pxMaruMinyaHangul) のライセンス状態をSIL OFL単体にしてもらう目的で、急遽作成しました
- 改修と縦書きに対応した Version 2.000 が完成次第、更新されます。バージョニングはこの仕様に備え、1.500となっています
